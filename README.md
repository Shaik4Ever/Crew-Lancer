# CREWLANCER Hiring

A modern web-based volunteer recruitment and management platform developed for hackathons and college events. The platform simplifies volunteer registration, management, and data export through an easy-to-use interface.

## 🚀 Features

- Volunteer Registration Form
- Responsive User Interface
- Dark & Light Mode
- Live Form Validation
- Filter Volunteers by Domain
- Delete Volunteer Records
- Export Volunteer Data to CSV
- Local Storage Support
- About Section
- Past Work Showcase

## 🛠️ Technologies Used

- HTML5
- Tailwind CSS
- JavaScript
- Local Storage

## 📂 Project Structure

```
CREWLANCER-Hiring/
│── index.html
│── README.md
│── .gitignore
```

## ▶️ How to Run

1. Clone the repository

```bash
git clone https://github.com/Shaik4Ever/CREWLANCER-Hiring.git
```

2. Open `index.html` in your browser.

No installation is required.
