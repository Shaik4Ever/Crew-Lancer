# CREWLANCER Hiring

🚀 CREWLANCER Hiring is a modern web-based volunteer recruitment platform developed during a hackathon to simplify volunteer registration and management for college events.

## Features

- 📝 Volunteer Registration
- 🌙 Dark & Light Mode
- 📱 Responsive Design
- 💾 Local Storage
- 🔍 Filter Volunteers by Domain
- 🗑 Delete Volunteer Records
- 📄 Export Volunteer Data to CSV
- 🔔 Toast Notifications

## Technologies Used

- HTML5
- Tailwind CSS
- JavaScript
- Local Storage

## How to Run

1. Clone the repository

```
git clone https://github.com/Shaik4Ever/CREWLANCER-Hiring.git
```

2. Open `index.html` in your browser.
